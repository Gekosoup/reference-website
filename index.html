Naming Conventions

It is important to include naming conventions for all file names, paths and folders when organizing!

E.X. 1

Aglonquin > Year 1 > Sem 1 > Class Name > Week Name > Assignment

Commit Messages

It is important to save and create commit messages regularily. It is also crucial that you make them descriptive! Include things like:

  • What you've added, edited or deleted
  • The number of lines that were touched
  • What tags you've used

To start a command, type < >

To end a command, type < / >-->

Some tags are known as parent tags, meaning additional tags (children tags) will be inside it.

Indentation is important because it allows you to view code easier and helps organize efficiently.
