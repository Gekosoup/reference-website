    Main

    This is a main. To create, type - main with < >

    • The main element wraps around the main content of the page.
    • Only used once per page

    Section

    This is a section. To create, type - section with < >

    • The main element wraps around the main content of the page.
    • Each section usually has its own heading

    Aside

    This is aside. To create, type - aside with < >

    The aside elenet represents a portion of a document where the content only relates to the documents main content. Usually shown as a sidebar on the website
